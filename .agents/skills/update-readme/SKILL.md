# update-readme

## When to use
Use when setup, structure, commands, or local developer workflow have changed.

## Output
- README sections to add or update
- exact commands
- concise explanation of each section

## Rules
- Keep the README practical.
- Reflect only the current repository state.
- Do not document future features as already implemented.
